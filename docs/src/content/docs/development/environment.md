# 開発環境

このリポジトリは、拡張本体を WXT、ドキュメントを Astro Starlight で管理します。

## 拡張本体

```bash
pnpm dev
```

## ドキュメント

```bash
pnpm docs:dev
```

## ビルド

```bash
pnpm build
pnpm docs:build
```

## チェック

```bash
pnpm check
pnpm docs:build
```

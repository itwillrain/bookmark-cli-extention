# プライバシーポリシー

Bookmark CLI Extensionは、Chromeのブックマークを疑似CLIから検索、起動、保存、整理するためのブラウザ拡張です。

このページでは、拡張機能が扱うユーザーデータ、利用目的、保存場所を説明します。

## 収集または利用するデータ

拡張機能は、機能提供のために次のデータをブラウザ内で扱います。

* Chromeのブックマークのタイトル、URL、フォルダ構造
* Chromeの閲覧履歴のタイトル、URL、アクセス時刻
* 現在のタブのタイトルとURL
* 拡張機能内で入力したコマンド履歴
* 拡張機能の設定、現在ディレクトリ、仮想タグ、ブックマーク利用統計

## 利用目的

これらのデータは、次の目的だけに利用します。

* ブックマークを検索して開く
* 現在のタブをブックマークとして保存する
* ブックマークやフォルダを整理する
* Chromeの閲覧履歴を検索候補として表示する
* コマンド履歴、現在ディレクトリ、設定、仮想タグ、利用統計を復元する

## 保存場所

拡張機能の設定、コマンド履歴、現在ディレクトリ、仮想タグ、利用統計は、Chrome拡張機能の`chrome.storage.local`に保存します。

一時的な起動元タブ情報は、Chrome拡張機能の`chrome.storage.session`に保存します。

ブックマーク本体と閲覧履歴はChromeが管理するデータを読み取り、必要な操作をユーザーの指示に応じて実行します。

## 第三者提供

拡張機能は、ユーザーデータを外部サーバーへ送信しません。

拡張機能は、ユーザーデータを販売しません。

拡張機能は、信用力の判断や融資目的でユーザーデータを利用しません。

## リモートコード

拡張機能は、パッケージ外のリモートJavaScriptまたはWasmを実行しません。

## このドキュメントで整えること

Bookmark CLI Extension は、ブラウザに保存された Bookmark と CLI の操作体験を接続するための拡張です。

まずは目的、設計方針、開発環境を小さく整理し、実装が進むたびにドメインモデルとユースケースを追記していきます。

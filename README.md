# Bookmark CLI Extension

Bookmark CLI Extension is a browser extension for managing Chrome bookmarks through a terminal-like command interface.

The project treats bookmarks as a filesystem-like workspace: search with `find`, open with `go`, move with `cd`, save the current tab with `mark`, and organize bookmark folders from a dedicated extension page.
